Summary: City Latitude & X

The purpose of this project was to analyze and understand the relationship that a City's Latitude has with the change of weather as you approach the equator. We used an API for OpenWeatherMap to pull data for apprx. 500 cities across the globe.


This site provides the dataset used, as well as visualizations based on the dataset to examine Latitude as it relates to Max Temperature, Humidity, Cloudiness, and Wind Speed using Matplotlib.
